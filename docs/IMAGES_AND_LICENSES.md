# 명소 카드 이미지와 저작권

## 왜 구글맵 이미지를 쓰지 않나요?

- **Google Maps**의 지도 타일, 스트리트뷰, 위성 화면 등은 **Google 이용약관**상 무단 복제·재배포·상업적 2차 이용이 제한됩니다.
- API를 쓰더라도 **표시 방식·저장·캐시** 조건이 따로 있어, 여행 플래너 UI에 임의로 붙이기 어렵습니다.

## 앱에서 쓰는 출처

1. **Unsplash** (`images.unsplash.com`)  
   - [Unsplash License](https://unsplash.com/license): 무료 사용·유사 CC0에 가깝게 널리 허용.  
   - 명소 카드의 **기본·폴백** 이미지로 사용(풍경 예시). 실제 촬영지가 해당 명소와 다를 수 있음.

2. **Wikimedia Commons**  
   - CC·퍼블릭 도메인 등 **라이선스가 파일마다 명시**됨.  
   - LLM이 **검증된 파일 URL**을 줄 수 있으면 사용 가능. 링크가 깨지면 클라이언트에서 Unsplash로 폴백.

3. **picsum.photos**  
   - 이전에 쓰던 예시 도메인은 일부 네트워크에서 차단되기도 해 **Unsplash로 통일**하는 방향을 권장합니다.

## 사용자가 직접 넣을 때

- 공식 관광청·명소 홈페이지의 **허용된 미디어 키트** 또는 **Commons/Unsplash**처럼 라이선스가 분명한 이미지를 사용하세요.
